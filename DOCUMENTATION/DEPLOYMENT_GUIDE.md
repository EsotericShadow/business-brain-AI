# Deployment Guide

This document provides instructions for deploying the application to Vercel, including the necessary environment variables.

## 1. Vercel Configuration

A `vercel.json` file is included in the root of this project. It is pre-configured to handle the monorepo structure, building both the frontend and backend and routing requests appropriately.

To deploy, simply connect your Git repository to Vercel and it will use this file automatically.
